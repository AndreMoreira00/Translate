# 🎧 Translate - Speech-to-English Translation

![Python](https://img.shields.io/badge/Python-3776AB?style=for-the-badge&logo=python&logoColor=white)

## 📝 Description

**Translate** is a Python-based project that captures a user’s speech and translates it into English in real-time. It provides a straightforward solution for translating spoken language into English, making it useful for communication across different languages. The project leverages Python libraries to convert speech into text and translate it into English.

## 🚀 Technologies Used

- **Python**: The programming language used to develop the project.
- **Speech Recognition Libraries**: To capture and convert user speech into text.
- **Translation Libraries**: For translating the captured text into English.

## 🌟 Features

- Real-time speech recognition from the user
- Instant translation of spoken language to English
- Console-based interface for simplicity

## 📈 How to Use

1. Clone this repository: `git clone https://github.com/AndreMoreira00/Translate.git`
2. Install the required dependencies: `pip install -r requirements.txt`
3. Run the main script: `python translate.py`
4. Speak into your microphone, and the spoken words will be automatically translated to English and displayed.

> **Note:** This project is designed to demonstrate speech-to-text and translation capabilities in Python, useful for language learning or communication purposes.
